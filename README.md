# AI CV Screening System - Premium

Hệ thống sàng lọc CV tự động bằng AI, kết hợp n8n Webhook và giao diện Glassmorphism hiện đại.

## Tính năng
- Phân tích hàng loạt hồ sơ (PDF, DOCX).
- Chấm điểm và nhận xét bằng AI.
- Chế độ Test Mode cho phép kiểm tra n8n Webhook dễ dàng.
- Giao diện Premium, mượt mà, thân thiện với mobile.

## Triển khai
1. Đưa lên Vercel hoặc GitHub Pages.
2. Cấu hình n8n Webhook URL trong file `index.html`.
